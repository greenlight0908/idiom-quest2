# 사자성어 여행단

안산 사리역 인근을 배경으로 진행되는 16비트 스타일 사자성어 학습 RPG입니다.

## 실행

저장소 폴더에서 간단한 로컬 웹 서버를 실행합니다.

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000`을 열면 게임을 시작할 수 있습니다.

## 주요 장면

- 시대정신 스터디카페와 충장로 거리
- 하오츠와 문학 작가 캐릭터
- 시대정신학원 로비 및 교실
- 한자가 새겨진 한자성어 나라와 퀴즈 석상
- 동산고 남학생·여학생 주인공

게임 본편은 `index.html`, 오프라인 캐시는 `sw.js`, 이미지 자산은 `assets/`에서 관리합니다.
